**Result:** Regional Finalist

##  Project Overview
AlumConnect is a national-level platform designed to bridge the communication gap between current university students and successful alumni. The goal was to create an intuitive, accessible, and highly scalable interface that encourages mentorship, job referrals, and community building.

## 🧠Design Thinking & Engineering Alignment
Because of my strong background in Computer Science and Machine Learning, every component in this Figma file was designed with **engineering viability** in mind.
I ensured that all UI components (buttons, cards, forms) could be easily translated into reusable React/Tailwind code by the frontend development team, maintaining strict grid systems and CSS-friendly styling.
